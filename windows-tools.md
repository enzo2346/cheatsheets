```shell
compmgmt.msc
```

```shell
taskschd.msc
```

```shell
eventvwr.msc
```

```shell
fsmgmt.msc
```

```shell
lusrmgr.msc
```

```shell

```

```shell
devmgmt.msc
```

```shell
diskmgmt.msc
```

```shell
services.msc
```

```shell

```

```shell
secpol.msc
```

